# The TinyBECC Project #

## Goals ##
TinyBECC implements the ISA SP100.11a Asymmetric Elliptic Curve Key Agreement

  * In C code
  * Minimizing code size while limiting performance sacrifice

## Algorithms ##

TinyBECC implements the following algorithms
  * AES
  * AES-MMO
  * SHA256
  * ECQV
  * ECMQV
  * ECPVS
  * ECDSA
